# Homeserver Dashboard

Легкий PHP-дашборд для домашних сервисов: карточки ссылок, иконки, категории, drag&drop и офлайн-ориентированный интерфейс.

## Возможности

- карточки сервисов с переходом по клику
- категории (создание, переименование, удаление, сворачивание)
- drag&drop:
  - карточек между категориями
  - порядка карточек внутри категории
  - порядка категорий
- автоопределение иконок из стандартных источников сайта
- ручная загрузка иконки
- поиск иконок по базе `Iconify`
- локальный кэш иконок в `icons-cache`
- хранение данных в `links.json`

## Требования

- PHP 8.0+ (допустимо 7.4+, но рекомендуется 8.x)
- веб-сервер (Nginx/Apache/Caddy) или встроенный PHP-сервер
- права на запись в:
  - `links.json`
  - директорию `icons-cache/`

## Быстрый запуск локально

```bash
php -S 0.0.0.0:8080
```

Открыть: `http://<IP_сервера>:8080`.

## Структура

- `index.php` — приложение (backend + frontend)
- `links.json` — база данных категорий и карточек
- `icons-cache/` — локально сохраненные иконки
