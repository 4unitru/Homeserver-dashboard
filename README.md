# Homeserver dashboard

Легкий self-hosted дашборд для домашних и LAN-сервисов на PHP: карточки, категории, drag&drop, управление иконками и офлайн-устойчивый UI.

## Ключевые возможности

- кликабельные карточки сервисов
- категории:
  - создание, переименование, удаление
  - сворачивание/разворачивание
  - drag&drop порядка категорий
- карточки:
  - drag&drop между категориями
  - drag&drop порядка внутри категории
  - сохранение layout (категория + порядок)
- единая модалка редактирования карточки:
  - изменение названия
  - иконка: ручная загрузка, поиск по `Iconify`, сброс на автоопределение
- автоопределение иконок сайта + локальный кэш в `icons-cache`
- интерфейс без CDN-зависимостей (базово работает без интернета)

## Технологии

- PHP (single-file app в `index.php`)
- JSON-хранилище (`links.json`)
- чистый CSS/JS без внешних UI-фреймворков

## Требования

- PHP 8.0+ (рекомендуется 8.2+)
- веб-сервер (Nginx/Apache/Caddy) или `php -S`
- права записи для процесса веб-сервера в:
  - `links.json`
  - `icons-cache/`

## Быстрый локальный запуск

```bash
php -S 0.0.0.0:8080
```

Открыть в браузере:

`http://127.0.0.1:8080` или `http://<LAN-IP>:8080`

## Структура проекта

- `index.php` — backend + frontend
- `links.json` — категории/карточки/порядок
- `icons-cache/` — локальный кэш иконок
- `SERVER_SETUP.md` — установка на сервер и права
- `UI_SOURCE_OF_TRUTH.md` — соглашения по UX/UI и механикам

## Развертывание

См. подробную пошаговую инструкцию в `SERVER_SETUP.md`.
